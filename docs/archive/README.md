# Archive

This directory stores historical project materials moved out of the repository
root to keep the executable codebase easier to scan.

Layout:

```text
docs/archive/
  prompts/   # historical Claude Code prompts and implementation instructions
  reports/   # historical experiment reviews and decision reports
  research/  # research recap and paper idea notes
```

Archived files are retained for traceability. They should not be used as the
default task entry point for new automated experiments.
